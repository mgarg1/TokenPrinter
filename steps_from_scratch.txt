1. downloaded the buster lite image from raspi.org
2. Burned the Image using Etcher for Windows (got from the same source)
3. Memory card will have 2 partitions now - i.boot partition (Fat32) ii.Actual OS (Ext4)
* In Windows - only boot partition will be shown


4. I created 2 files in boot partition:
i.  ssh - empty content
ii. wpa_supplicant.conf 

ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
country=IN

network={
 ssid="<Name of your WiFi>"
 psk="<Password for your WiFi>"
}

Ref: https://www.raspberrypi.org/documentation/configuration/wireless/headless.md

5. Boot up the raspi(and waited for sometime) and checked its entry in the router's Ip table to find out its ip (knowing their MAC in advance will also help)

6. ssh pi@192.168.0.102 (ip found from the previous step)
7. sudo apt-get update; sudo apt-get -y upgrade
8. raspi-config -> Interface settings -> enable VNC
9. raspi-config -> Localisation Options -> Change Timezone -> Asia -> Kolkata

10. Setup Real Time Clock - DS3231
i. Connect the I2C pins of RTC to pin 3(GPIO 8),5(GPIO 9) of rPi, Vcc and Gnd to pin 1,7 of rPi
ii. sudo nano /etc/modules # and append rtc-ds1307
iii. sudo apt-get install i2c-tools
iv. sudo i2cdetect -y 1
v. sudo nano /etc/rc.local # and append before 'exit 0'
echo ds1307 0x68 > /sys/class/i2c-adapter/i2c-1/new_device
hwclock -s
vi. sudo reboot
vii. check date
viii. sudo hwclock -w

Ref: https://www.raspberrypi-spy.co.uk/2015/05/adding-a-ds3231-real-time-clock-to-the-raspberry-pi/


11. sudo apt-get install git #Install git
12. git clone https://github.com/mgarg1/TokenPrinter.git
https://python-escpos.readthedocs.io/en/latest/user/raspi.html

13. sudo apt-get install virtualenv
14. cd TokenPrinter; virtualenv venv
15. source venv/bin/activate
16. give permission to the user for usb:
sudo groupadd usbusers
sudo usermod -a -G usbusers pi
sudo touch /etc/udev/rules.d/99-usbusers.rules

# put inside /etc/udev/rules.d/99-usbusers.rules
SUBSYSTEM=="usb", GROUP="usbusers", MODE="0660"
SUBSYSTEMS=="usb", GROUP="usbusers", MODE="0660"

#restart
sudo init 6
Ref:https://www.odoo.com/documentation/user/9.0/point_of_sale/overview/setup.html

13. sudo apt-get install python3-pip
14. python3 -m pip install -r requirements.txt
15. Connect the printer to another USB port of rPi zero
i. $ lsusb
Bus 001 Device 002: ID 0456:0808 Analog Devices, Inc.

15. python3 main.py
